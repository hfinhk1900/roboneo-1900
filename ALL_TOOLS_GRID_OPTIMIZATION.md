# All Tools 卡片布局优化

## 📋 优化概述

优化了 Home Page 的 All Tools 区块的卡片网格布局，确保卡片始终撑满页面，保持 1 行或 2 行的整齐排列。

---

## ✅ 布局变更

### 之前的布局

```tsx
// 会导致第二行只有 1 个卡片，不整齐
grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-5
```

**问题**：
- ❌ 在大屏幕（lg）上：第一行 5 个，第二行 1 个（不撑满）
- ❌ 卡片分布不均匀
- ❌ 视觉不平衡

### 现在的布局

```tsx
// 6 个卡片完美分布
grid-cols-1 sm:grid-cols-2 md:grid-cols-3
```

**优势**：
- ✅ **移动端（<640px）**: 1 列，6 行 - 垂直排列
- ✅ **平板（640px-768px）**: 2 列，3 行 - 完美撑满
- ✅ **桌面（≥768px）**: 3 列，2 行 - 完美撑满
- ✅ 所有卡片大小一致
- ✅ 视觉平衡美观

### 如果未来增加/减少工具

| 工具数量 | 推荐布局 | 效果 |
|---------|---------|------|
| 3 个 | `md:grid-cols-3` | 1 行 × 3 列 ✅ |
| 4 个 | `md:grid-cols-2` | 2 行 × 2 列 ✅ |
| 5 个 | `md:grid-cols-5` | 1 行 × 5 列 ✅ |
| 6 个 | `md:grid-cols-3` | 2 行 × 3 列 ✅ (当前) |
| 8 个 | `md:grid-cols-4` | 2 行 × 4 列 ✅ |
| 9 个 | `md:grid-cols-3` | 3 行 × 3 列 ✅ |

**原则**：
- 选择能被工具总数整除的列数
- 确保最后一行也撑满
- 避免出现单个卡片占一行的情况
